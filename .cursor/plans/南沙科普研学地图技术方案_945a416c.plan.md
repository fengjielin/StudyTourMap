## 项目实施计划

### 技术栈


| 类别   | 技术选型                               |
| ---- | ---------------------------------- |
| 框架   | Vue 3 + Vite + TypeScript          |
| 地图   | Leaflet + @vue-leaflet/vue-leaflet |
| 状态管理 | Pinia                              |
| 样式   | SCSS + CSS Variables               |
| 动画   | CSS Keyframes                      |
| 图标   | Lucide Vue                         |


### 主题色配置

```css
:root {
  --primary: #2563EB;     /* 科技蓝 - 主色 */

}
```

### 1. 项目初始化

- 创建 Vue 3 + Vite + TypeScript 项目
- 安装依赖：leaflet, @vue-leaflet/vue-leaflet, pinia, sass, lucide-vue-next
- 配置项目结构

### 2. 核心工具模块 `src/core/`

**TileProvider.ts** - 瓦片源管理器

```typescript
export const tileProviders = {
  gaode: {
    name: '高德地图',
    subdomains: ['1', '2', '3', '4'],
    normal: 'https://webst0{s}.is.autonavi.com/3.0版/styles/light/{z}/{x}/{y}',
    satellite: 'https://webst0{s}.is.autonavi.com/3.0版/styles/satellite/{z}/{x}/{y}',
  },
  baidu: {
    name: '百度地图',
    normal: 'https://maponline{0-3}.bdimg.io/stylenormal?qt=tile&x={x}&y={y}&zoom={z}',
    satellite: 'https://maponline{0-3}.bdimg.io/stylesatellite?qt=tile&x={x}&y={y}&zoom={z}',
  },
  tianmap: {
    name: '天地图',
    subdomains: ['0', '1', '2', '3', '4', '5', '6', '7'],
    normal: 'https://t{s}.tianditu.gov.cn/DataServer?T=vec_w&x={x}&y={y}&l={z}',
    label: 'https://t{s}.tianditu.gov.cn/DataServer?T=cva_w&x={x}&y={y}&l={z}',
  },
  osm: {
    name: 'OpenStreetMap',
    normal: 'https://tile.openstreetmap.org/{z}/{x}/{y}',
  }
}
```

**CoordTransform.ts** - 坐标系转换（WGS-84 ↔ GCJ-02 ↔ BD-09）

**GeoCoder.ts** - 地址地理编码（通过高德API获取坐标）

### 3. 地图组件 `src/components/Map/`


| 组件                   | 说明              |
| -------------------- | --------------- |
| **MapContainer.vue** | 地图容器，集成 Leaflet |
| **BaseMarker.vue**   | 圆形标记点，带呼吸灯闪烁动画  |
| **MapSwitcher.vue**  | 地图源切换下拉控件       |


### 4. 信息展示 `src/components/Panel/`

**BaseInfoPanel.vue** - 基地详情面板

- 基地名称、所属单位、研学路线名
- 详细地址
- 图片展示区（占位图，后续替换）
- 关闭按钮

### 5. 数据层 `public/data/bases.json`

基于 [docs/研学基地.md](docs/研学基地.md) 的10个基地数据，结构如下：

```json
[
  {
    "id": "base_01",

    "routeName": "「逐梦空天·未来先锋」高精尖科技研学营",
    "address": "广州市南沙区前知路2号",

    "images": []
  }
```

**注意**：坐标通过地理编码自动获取，用户可后续调整。

### 6. 动画效果

**圆形标记闪烁动画**：

```css
@keyframes pulse {
  0%, 100% {
    transform: scale(1);
    opacity: 1;
  }
  50% {
    transform: scale(1.2);
    opacity: 0.7;
  }
}
```

### 7. UI 设计

- 简洁现代的卡片式设计
- 悬浮式地图控件（右上角）
- 半透明信息面板（底部滑出）
